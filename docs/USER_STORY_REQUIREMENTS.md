# User Story Requirements

## Non-Admin User
- As a non-admin user, I want to use the application that is not exposed on the internet
- As a non-admin user, I want to use the application on MacOS and Linux devices
- As a non-admin user, I want to manage the DOR
- As a non-admin user, I want to manage the RTI

### Non-Admin User - Manage Identities
- As a non-admin user, I want to update my profile

## Non-Admin User - Manage Data Object Repository (DOR)
- As a non-admin user, I want to add a Data Object into the DOR
- As a non-admin user, I want to remove a Data Object from the DOR
- As a non-admin user, I want to grant access to a Data Object from the DOR
- As a non-admin user, I want to revoke access to a Data Object from the DOR

## Non-Admin User - Manage Processor
- As a non-admin user, I want to build processor image from a local source
- As a non-admin user, I want to build processor image from a Github source

---

## Admin User
- As a admin user, I want to use the application that is not exposed on the internet
- As a admin user, I want to use the application on MacOS and Linux devices
- As a admin user, I want to manage the RTI
- As a admin user, I want to manage the monitor nodes

### Admin User - Manage Identities
- As a admin user, I want to create new keystore/identity
- As a admin user, I want to see a list of keystores/identities
- As a admin user, I want to update my profile
- As a admin user, I want to remove keystores/identities from the list
- As a admin user, I want to see details of each keystores/identities
- As a admin user, I want to see a list of all identities known to a node
- As a admin user, I want to publish an identity to a node
- As a admin user, I want to manage credentials

## Admin User - Manage Data Object Repository (DOR)
- As a admin user, I want to add a Data Object into the DOR
- As a admin user, I want to remove a Data Object from the DOR
- As a admin user, I want to grant access to a Data Object from the DOR
- As a admin user, I want to revoke access to a Data Object from the DOR

## Admin User - Manage Processor
- As a admin user, I want to build processor image from a local source
- As a admin user, I want to build processor image from a Github source
